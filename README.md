Lista de Exercícios 3 - JavaScript Básica

Esta é a terceira e última lista de exercícios de lógica de programação usando JavaScript e Node.js.
O objetivo dessa lista foi aprender e diferenciar as estruturas de repetição usando for, while e do while.

Exercícios Resolvidos:

Exercício 1: Calcular e mostrar a soma de todos os números de 1 até 100 usando for.

Exercício 2: Calcular o fatorial de um número inteiro positivo usando for.

Exercício 3: Bloquear o acesso enquanto a senha digitada for errada usando while.

Exercício 4: Fazer uma contagem regressiva até zero usando while.

Exercício 5: Somar vários números e parar o programa assim que o número zero for digitado usando while.

Exercício 6: Mostrar um menu de banco na tela até o usuário escolher a opção de sair usando do while.

Exercício 7: Continuar pedindo uma nota de prova até que seja digitado um valor válido entre 0 e 10 usando do while.

Exercício 8: Ficar repetindo a execução do programa enquanto o usuário digitar S usando do while.

Exercício 9: Calcular a média de vários números positivos até o usuário responder que NÃO quer continuar usando do while.

Como rodar os códigos:

Para testar no terminal do VS Code usando o Node, é só digitar:

node .\exe01.js
